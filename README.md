# MCP 项目集主索引

本项目集旨在整合一系列基于 Model Context Protocol (MCP) 的工具。当前，各个独立的 MCP 工具作为不同的子项目，分别存放于独立的分支中进行开发与管理。本 `main` 分支的 README 作为中心索引，提供所有子项目的统一入口和导航。

## 🚀 快速导航

1. **查看所有可用分支**：您可以在仓库的 “Branches” 页面或通过 `git branch -a` 命令查看完整的项目分支列表。
2. **切换到目标项目**：使用 `git checkout <branch-name>`（例如 `git checkout mcp-diskmanager`）来切换到特定项目的开发环境。
3. **查阅项目文档**：切换至对应分支后，请查看该分支根目录下的 `README.md` 文件，以获取该项目的详细功能介绍、安装指南、配置说明和使用教程。

## 🔧 通用约定

为了保持项目集结构清晰与风格统一，所有子项目的 README 建议遵循以下核心结构模板：

* **项目名称与标语**：清晰的项目名和一句话简介。
* **功能特性**：以列表形式展示核心功能与技术亮点。
* **快速开始**：包含环境要求、安装步骤和能让用户快速运行起来的基本使用示例。
* **详细使用指南**：提供更全面的配置说明、命令行选项、API 文档等。
* **贡献指南**：说明代码规范、提交流程及如何报告问题。
* **许可证**：明确项目的开源许可证。
